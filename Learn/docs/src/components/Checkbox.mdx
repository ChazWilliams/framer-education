# Checkbox

A checkbox can be on or off. It can be valid or not, depending on its validation function.

## Props

| Prop          | Type     | Default                | Notes                                           |
| :------------ | :------- | :--------------------- | :---------------------------------------------- |
| value         | boolean  | false                  | Whether the Checkbox is checked.                |
| disabled      | boolean  | false                  |
| validation    | function | (value) => true        | A callback that returns `true` or `false`.      |
| onValueChange | function | (value, valid) => void | The callback to fire when checked or unchecked. |

## Examples

### Override

```tsx
import { Override } from "framer"

export function Checkbox(): Override {
	return {
		onValueChange: value => console.log("Checked: " + value),
	}
}
```

#### Validation

```tsx
import { Override } from "framer"

export function ValidWhenChecked(): Override {
	return {
		validation: value => value === true,
		onValueChange: value => console.log("Checked: " + value),
	}
}
```

### Code Component

#### Basic

```tsx
import * as React from "react"
import { Stack, Frame } from "framer"
import * as Learn from "@framer/steveruizok.education/dist/index"
const Checkbox = Learn.__framer__.sourceModules["./Checkbox.tsx"]().Checkbox

export const CheckboxTest = props => {
	const handleValueChange = (value, valid) => {
		console.log(`Value: ${value}, Valid: ${valid}`)
	}

	return (
		<Stack width="100%" height="100%" direction="horizontal">
			<span>Option</span> <Checkbox onValueChange={handleValueChange} />
		</Stack>
	)
}
```

#### Validation

```tsx
import * as React from "react"
import { Stack, Frame } from "framer"
import * as Learn from "@framer/steveruizok.education/dist/index"
const Checkbox = Learn.__framer__.sourceModules["./Checkbox.tsx"]().Checkbox

export const CheckboxTestValid = props => {
	const validation = value => value === true

	const handleValueChange = (value, valid) => {
		console.log(`Value: ${value}, Valid: ${valid}`)
	}

	return (
		<Stack width="100%" height="100%" direction="horizontal">
			<span>Option</span>{" "}
			<Checkbox validation={validation} onValueChange={handleValueChange} />
		</Stack>
	)
}
```
